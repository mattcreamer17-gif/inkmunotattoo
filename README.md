# Inkmunotattoo x Eduba landing page

A bespoke, conference-handoff landing page prepared for Inkmunotattoo (eMerge Americas 2026, Startup Spotlight SS70). Built in Spanish with an ES/EN toggle, since the founders are Spanish-first and the investor-facing copy needs English parity for US conversations. The page carries the fit-analysis pitch body verbatim (Opening, Frame, Precedent, Method, CTA) using Inkmunotattoo's brand tokens: ink-black on near-white, deep science blue for links and CTAs, Inter throughout, JetBrains Mono for the meta labels. A simple macrophage-cycle SVG motif replaces the placeholder logo since no official logo asset exists. Footer carries the only Eduba mark. CTA goes to Matt Creamer's Calendly.
